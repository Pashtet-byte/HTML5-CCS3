# Школа танцев «Ритм» — сайт на Bootstrap 5.3

## Цель проекта
Создание современного, адаптивного и информативного сайта для школы танцев. Сайт даёт посетителям полное представление о направлениях, расписании, преподавателях и контактах, а также позволяет быстро записаться на пробное занятие.

## Страницы
1. **Главная** — приветственный экран с каруселью, преимущества, превью направлений и расписания, форма пробного урока.
2. **Направления** — подробное описание танцевальных стилей, сгруппированных в аккордеон, карточки преподавателей.
3. **Расписание** — понедельное расписание во вкладках (табы), кнопка записи через модальное окно.
4. **Контакты** — адрес, телефон, email, соцсети и форма записи.

## Использованные компоненты Bootstrap
- **Navbar** — навигационное меню с логотипом и кнопками.
- **Carousel** — на главной для демонстрации фотографий.
- **Cards** — для преимуществ, направлений, преподавателей.
- **Accordion** — группировка направлений на странице «Направления».
- **Tabs** — переключение по дням в расписании.
- **Table** — отображение расписания.
- **Forms** — формы записи и пробного урока (с правильными `label`).
- **Buttons** — основные CTA-кнопки.
- **Modals** — окно записи и политика конфиденциальности.
- **Alerts** — информационное сообщение на странице расписания.
- **Badges** — метки уровней и категорий.
- **List group** — для списка фитнес-направлений.
- **Bootstrap Icons** — иконки для визуального выделения.

## Кастомизация (style.css)
- Добавлен фоновый градиент в hero-секцию.
- Уменьшен размер заголовка `display-5` на мобильных.
- Ограничена ширина круглых фотографий преподавателей.
- Глобально отключено горизонтальное переполнение (`overflow-x: hidden` для section).

## Адаптивность
Сайт корректно отображается на трёх контрольных точках (мобильные, планшеты, десктопы) благодаря встроенной сетке Bootstrap и медиа-запросам.

## Изображения
Все изображения заменены на локальные файлы из папки `assets/images/`.  
Для работы сайта необходимо поместить в эту папку файлы с именами, указанными в HTML (например, `anna.jpg`, `hiphop.jpg` и т.д.). Рекомендуется использовать изображения с соотношением сторон, близким к исходным placehold.it (примерно 1:1 для преподавателей и 4:3 для карточек направлений).
